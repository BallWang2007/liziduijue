## 一、项目概览（一句话说清）

- **项目名称**：离子对决（Ion Duel）  
- **介绍**：把酸碱中和、离子配对与“盐”的增益机制做成 1v1 回合制卡牌对弈：试剂下回合爆发、回合开始可中和反制，3–5 分钟一局，适合游园会现场试玩与科普展示。

## 二、为什么做（Why）

1) **受众是谁？**  
游园会现场观众（中学生/大学生/亲子），对“化学”有好奇心但不想做题；在展位停留时间短，需要“一眼能懂、立刻能玩”的互动。

2) **想创造什么？（痛点/体验）**  
把“抽象化学反应”变成可操作的对弈：观众通过拖拽卡牌完成配对合成，体验“预测下回合爆发”和“回合开始中和”的博弈爽点。

3) **现有方案还缺什么？**  
很多科普要么偏讲解、互动弱；要么偏娱乐、化学不真实。我们希望兼顾：**能解释、能对抗、能复盘**，让观众愿意再来一局。

## 三、做了什么（What）

> 核心设计功能（不超过 3 个）

### 功能 1：化学合成驱动的卡牌对弈
- **功能名称**：离子 → 酸/碱/盐 合成  
- **设计意图 / 解决的问题**：让化学规则成为“出牌逻辑”，玩家自然学会强/弱酸碱、离子配对。  
- **关键截图 / 流程**：手牌拖拽离子到实验台/另一离子 → 合成 → 送入反应槽（待补图）

### 功能 2：下回合爆发 + 回合开始中和窗口（强互动）
- **功能名称**：延迟爆发与中和反制  
- **设计意图 / 解决的问题**：让对局有预判与反制空间，避免“先手一波秒”。  
- **关键截图 / 流程**：对手反应槽下回合爆发 → 我方回合开始进入中和窗口（最多 2 张牌）→ 结算伤害（待补图）

### 功能 3：盐机制（自用增益 + 中和奖励）
- **功能名称**：盐（NaCl 等）与盐标记  
- **设计意图 / 解决的问题**：让“中和成功”有回报，同时给玩家更多策略分支。  
- **关键截图 / 流程**：  
  - 离子合成盐：Na⁺ + Cl⁻ → NaCl（占反应槽，下回合自用：回 2 / 抽 1）  
  - 完全中和 → 获得盐标记 → 出牌阶段可消耗转化为自用盐（待补图）

## 四、未来规划（如果晋级，下一步怎么做）

1) **更强的可玩性**：加入更多离子/反应（沉淀、氧化还原、缓冲等）做“扩展包”，并提供入门/标准/高手难度。  
2) **更好现场体验**：增加新手引导、对局回放/高光提示、现场排行榜（可离线记录）。  
3) **更广传播**：做单人对 AI 模式、卡组预设与教学关卡，让观众带走链接也能继续玩。
